API Project: URL Shortener Microservice

By Brandon Roth | Project GitHub

What is this?

This is an API project for freeCodeCamp's Back End Development Certification that accepts a valid URL as a parameter and returns a shortened URL in the JSON response. Visiting the shortened URL redirects to the original URL passed as a parameter.

Creation Usage:

https://brandonr-shorturl.herokuapp.com/new/https://www.google.com

Creation Output:

{"original_url":"https://www.google.com","short_url":"https://brandonr-shorturl.herokuapp.com/4188"}

Redirection Usage:

https://brandonr-shorturl.herokuapp.com/4188

Redirects to:

www.google.com
